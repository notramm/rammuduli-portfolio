A modern, interactive **3D developer portfolio** built with cutting-edge web technologies.  
Designed to deliver a visually immersive experience while maintaining performance and responsiveness.

---

## 🌐 Live Demo

👉 https://rammuduli.vercel.app/

---

## ✨ Features

- 🌌 Immersive **3D experience** using React Three Fiber
- 🎯 Smooth animations powered by GSAP
- ⚡ High performance & optimized rendering
- 🎨 Fully responsive design
- 🌗 Dark / Light theme support
- 🧠 State management using Zustand

---

## 🛠 Tech Stack

| Category | Tech |
|----------|------|
| Framework | Next.js (App Router) |
| UI | React |
| 3D | React Three Fiber, Drei |
| Animation | GSAP |
| State | Zustand |
| Styling | Tailwind CSS |
